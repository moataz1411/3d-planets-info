**3D Planets Info**

A simple interactive website built with Three.js that showcases 3D models of planets and a black hole.

**Features**
- Interactive 3D planet models
- Automatic rotation with user controls
- Educational information about each celestial object

**languages used**
- HTML
- CSS
- JavaScript

**Models**
- Mars
- Jupiter
- Saturn
- Uranus
- Black Hole
